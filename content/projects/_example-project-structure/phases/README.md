# Phases Directory

This folder contains the phases of your project.

## What Goes Here

Each phase is a major milestone or stage of your project:
- Discovery/Research phase
- Design phase
- Development phases
- Launch phase
- Post-launch iteration

## Structure

```
phases/
├── phase-1-discovery/
│   ├── overview.md      # Phase goals, success criteria, timeline
│   └── sprints/         # Sprints within this phase
├── phase-2-design/
│   ├── overview.md
│   └── sprints/
└── phase-3-development/
    ├── overview.md
    └── sprints/
```

## Phase Overview Should Include

- Phase name and number
- Goals and success criteria
- Timeline (start/end dates)
- Key milestones
- Dependencies on other phases
- Risks and mitigation

---

*Phases break large projects into manageable chunks with clear milestones.*
